# mikro-manager
command line manager for Mikrotik basic on python-rosapi by lblasc
https://github.com/lblasc/python-rosapi

to start: ./mikro.py 192.168.88.1 admin passwd

!! WARNING!!!
for empty password use ""


Few Example:

 - set ssid !#Your_new_ssid
 - print ssid
 - print hosts
 - help
 - quit
